# NexCode

NexCode — A futuristic multi-language web code editor.

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```

## Features

- Multi-language support
- Improved scroll-based UI
- User authentication and persistent storage

## Technologies Used

- Node.js & Express
- Vanilla JavaScript, HTML, CSS (Frontend)
